Question-1:What are the different ways to select an element in the DOM?

getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()

Question-2:What is the difference between innerHTML, innerText and textContent?

innerHTML: Returns/sets HTML with tags. innerText: Returns/sets only visible text, ignore hidden elements. textContent: Returns/sets all text, including hidden, without HTML tags.

Question-3:What is event delegation in the DOM?

Event Delegation is a technique where we don't add event listeners to each child individually. Instead, we add it to the parent, and the event listener works for all the children inside the parent.

Question-4:What is event bubbling in the DOM?

Event Bubbling is when an event occurs on an inner element and then "bubbles" up to its parent element. The event triggers first on the inner element, then moves to the parent.

Question-4:How do you create,add,and remove elements using JavaScript?

Create: document.createElement() Add: appendChild() Remove: removeChild() or remove()
